Graphic Synthesis Sequencer

1. Initialisierung & Zeichnung

  • Pattern-Zuweisung: Wähle in der Pattern Bank (unten rechts) eine leere Zelle, um eine neue Sequenz zu initiieren.
  • Grafische Eingabe: Die X-Achse repräsentiert die Zeit ($t$), die Y-Achse die fundamentale Frequenz ($f_0$). Zeichne Vektoren in das Canvas.
  • Werkzeuge: Wähle zwischen Draw (Vektor-Eingabe) und Erase (Vektor-Tilgung).

2. Spur-Architektur (Tracks)

  • Wellenformen: Definiert das harmonische Spektrum. ∿ (Sinus/Grundton), ⊓ (Rechteck/ungerade Obertöne), ∧ (Sägezahn/alle Obertöne).
  • Snap (Quantisierung): Erzwingt eine harte Rasterung der X-Koordinaten zur absoluten Taktsynchronität.
  • Routing: Nutze Solo/Mute für isolierte Audiosignal-Kontrolle und die Slider für die Amplituden-Skalierung.

3. Performative Interaktion

  • Poke Me Pad: Ein taktiles Echtzeit-Interface. Triggert sofortige Frequenz-Zuweisungen auf der selektierten Spur.
  • Effekt-Matrix: Klicke auf die Farbindikatoren, um das Audiosignal der jeweiligen Spur in dedizierte DSP-Module (Verzögerung, Hall, Filter etc.) zu routen.
  • X/Y Link: Wandelt das Pad in einen Makro-Controller um, der algorithmisch Parameter der Effekt-Engine interpoliert.

4. Taktung & Makro-Struktur

  • Transport: Start/Stopp der globalen Playback-Schleife (Shortcut: Leertaste).
  • Harmonize (Tonleiter-Rastung): Zwingt alle Y-Koordinaten durch mathematische Rundung auf vordefinierte musikalische Skalen.
  • EXT SYNC: Synchronisiert den internen Timer via Web-MIDI-API mit externen Clocks (DAWs/Hardware). Erfordert virtuelle MIDI-Treiber.

5. Geometrische Synthese-Werkzeuge (Brushes)

  • Standard: Kontinuierlicher Phasen-Oszillator. Übersetzt die Vektoren in absolut stufenlose Frequenzverläufe (perfektes Portamento).
  • Pressure: Geschwindigkeitsabhängige Linienführung. Simuliert haptischen Druck durch Analyse der Maus-Vektorenlänge.
  • Calligraphy: Winkel-determinierte Geometrie. Erzeugt statische Amplitudenbänder, inspiriert durch historische grafische Notation.
  • Particles: Stochastische Mikromontage (Granularsynthese). Zerlegt die Kurve in asynchrone, granulare Millisekunden-Grains – erzeugt texturierte, prasselnde Klangwolken.
  • Chord: Harmonische Frequenz-Vervielfachung. Generiert parallele Oszillatoren, die an strikte musikalische Ratios (Intervalle) gekoppelt sind. Konfigurierbar im Sub-Menü.
  • Xenakis Swarm: Unkorrelierte Frequenzmodulation. Zeichnet fünf Linien, deren Tonhöhen durch gegenläufige LFO-Algorithmen gegeneinander verstimmt werden, was dichte, schwebende Cluster erzeugt.
  • Fractal: Destabilisierter Synthesizer.
    DSP-Steuerung via "FRACTAL" Rack:
    → CHAOS: Injiziert Rausch-Jitter in die Y-Matrix, was die Grundfrequenz stochastisch zerrüttet.
    → MORPH: Aktiviert eine nicht-lineare Transferfunktion (WaveShaper). Erzeugt extreme harmonische Verzerrung (Wavefolding) und visualisiert das Signal-Clipping rot.

6. Datensicherung & Rendering

  • SAVE: Speichert den Zustand aller Vektor-Matrizen auf ein Pad der Bank.
  • Live Arrangement: Sequenzielles Umschalten zwischen Pattern-Speichern in Echtzeit ohne Phasen-Unterbrechung.
  • Export WAV: Berechnet die Audio-Buffer iterativ (Offline-Rendering) und liefert verlustfreies PCM-Audio.
  • Live REC: Rekordiert den Master-Bus und exportiert improvisierte Performances.
  • Import/Export: Serialisiert die komplette Session (JSON) für späteren Recall.

Graphic Synthesis Sequencer

1. Initialization & Drawing

  • Assign Pattern: Select an empty cell in the Pattern Bank (bottom right) to initialize a new loop sequence.
  • Vector Input: The X-axis represents time ($t$), the Y-axis the fundamental frequency ($f_0$). Draw vectors into the canvas.
  • Tools: Toggle between Draw (vector insertion) and Erase (vector deletion).

2. Track Architecture

  • Waveforms: Defines the harmonic spectrum. ∿ (Sine/Fundamental), ⊓ (Square/Odd Harmonics), ∧ (Sawtooth/All Harmonics).
  • Snap (Quantization): Forces absolute temporal synchronization by locking coordinates to a grid.
  • Routing: Use Solo/Mute for isolated audio monitoring and sliders for amplitude scaling.

3. Performative Interaction

  • Poke Me Pad: A tactile real-time interface. Triggers instantaneous frequency allocations on the targeted track.
  • FX Matrix: Click the track-color indicators to route the audio signal into dedicated DSP modules (Delay, Reverb, etc.).
  • X/Y Link: Hijacks the tactile pad to act as a macro-controller, algorithmically interpolating FX parameters.

4. Clock & Macro Structure

  • Transport: Initiates or halts the global playback loop (Shortcut: Spacebar).
  • Harmonize: Algorithmic pitch-correction. Constrains all Y-coordinates to predefined musical scales.
  • EXT SYNC: Uses the Web MIDI API to slave the internal timer to external hardware or DAWs. Requires virtual MIDI ports.

5. Geometric Synthesis Brushes

  • Standard: Continuous phase oscillator. Translates vectors into flawless frequency sweeps (Portamento).
  • Pressure: Velocity-dependent strokes. Simulates pressure by analyzing the vector distance of your mouse movements.
  • Calligraphy: Angle-determined geometry. Creates static amplitude bands inspired by historical graphic notation.
  • Particles: Stochastic micro-montage (Granular Synthesis). Disintegrates the curve into asynchronous, millisecond-long grains, creating textured sound clouds.
  • Chord: Harmonic frequency multiplication. Generates parallel oscillators locked to strict musical ratios. Defined via the sub-menu.
  • Xenakis Swarm: Uncorrelated frequency modulation. Draws five parallel lines whose pitches are dynamically detuned by intersecting low-frequency oscillators (LFOs).
  • Fractal: Destabilized synthesizer.
    DSP-Control via "FRACTAL" Rack:
    → CHAOS: Injects noise-jitter into the Y-matrix, causing microtonal frequency scrambling.
    → MORPH: Activates a non-linear transfer function (WaveShaper). Induces extreme harmonic distortion (Wavefolding) and highlights the line's clipping in red.

6. Serialization & Rendering

  • SAVE: Commits the current state of all vector matrices to a pad in the memory bank.
  • Live Arrangement: Sequentially switch between stored memory banks in real-time without phase interruptions.
  • Export WAV: Iteratively renders the audio buffer offline, providing a mathematically flawless PCM audio file.
  • Live REC: Records the master bus output directly to a downloadable file.
  • Import/Export: Serializes the entire session (JSON) for later recall.
