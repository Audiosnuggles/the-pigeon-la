The Pigeon: Referenzhandbuch

The Pigeon ist ein webbasiertes System für grafische Vektorsynthese, das zweidimensionale Vektordaten in Echtzeit in Steuersignale für komplexe Audio-Engines übersetzt. Das Interface fungiert als akustisches Koordinatensystem: Die X-Achse definiert den Zeitverlauf ($t$), die Y-Achse die fundamentale Frequenz ($f_0$). Die visuelle Darstellung dient dabei als abstrakte grafische Partitur und nicht als Abbildung der physikalischen Schallwelle.

Inspiriert von elektroakustischen Pionierarbeiten (wie Daphne Orams „Oramics“) und konzipiert als performatives Instrument, bildet das System einen methodischen Kontrast zur funktionalen Dichte herkömmlicher DAWs. Der bewusste Verzicht auf etablierte Raster reduziert die Komposition auf eine unmittelbare, visuelle Eingabe und begünstigt experimentelle Ansätze der Klangformung.

Snap

Routet Audiosignale zielgerichtet in dedizierte DSP-Module (Farbindikatoren).

Kontinuierlicher Phasen-Oszillator (ideal für Portamento-Gesten).

The Pigeon: Reference Manual

The Pigeon is a web-based system for graphical vector synthesis that translates two-dimensional vector data into control signals for complex audio engines in real time. The interface functions as an acoustic coordinate system: the X-axis defines the timeline ($t$), and the Y-axis defines the fundamental frequency ($f_0$). The visual representation serves as an abstract graphical score rather than a depiction of a physical sound wave.

Inspired by early electroacoustic works (such as Daphne Oram's "Oramics") and designed as a performative instrument, the system serves as a methodical contrast to the functional density of conventional DAWs. By intentionally bypassing established grids, the process is reduced to immediate visual input, fostering experimental approaches to sound design.

Architecture & Routing

Pattern Bank
Select an empty memory cell (A-C) to initiate a new sequence.
Waveforms
Defines the spectrum: ∿ (Sine), ⊓ (Square), ∧ (Sawtooth).
Snap
Forces hard quantization on the X-axis for absolute synchronization.
FX Matrix
Routes audio signals into dedicated DSP modules using color indicators.

Performance & Macro

Poke Me Pad
A tactile interface for instant frequency triggers on the target track.
X/Y Link
Temporarily hijacks the pad to act as a macro controller for FX parameters.
Harmonize
Constrains all drawn Y-coordinates to mathematical musical scales.
Ext Sync
Synchronizes the global playback loop to an external DAW via MIDI.

Synthesis Engines (Brushes)

Standard
Continuous phase oscillator (ideal for portamento gestures).
Pressure
Velocity-dependent stroke thickness, simulating physical pressure.
Particles
Stochastic micro-montage (real-time granular synthesis).
Chord
Harmonic frequency multiplication based on defined intervals.
Xenakids
Uncorrelated phase modulation generating dense dissonance clusters.
Rorbach
Axis-symmetric counterpoint (mirrors the Y-axis).
Overtone
Additive spectral synthesis by injecting logarithmic partials.
DX7 (FM)
Two-operator frequency modulation (John Chowning method).
Fractal
Destabilized synthesizer. Controlled via the FRACTAL rack (Chaos = Noise Jitter, Morph = Wavefolding intensity).

The Pigeon: Manuel de Référence

The Pigeon est un système web de synthèse vectorielle graphique qui traduit des données bidimensionnelles en signaux de contrôle pour des moteurs audio en temps réel. L'interface fonctionne comme un système de coordonnées acoustiques : l'axe X définit le temps ($t$), l'axe Y la fréquence fondamentale ($f_0$). La représentation visuelle sert de partition abstraite et non de représentation de l'onde sonore.

Inspiré par les travaux pionniers de l'électroacoustique et conçu comme un instrument performatif, le système constitue un contraste méthodique à la densité des DAW conventionnels. L'absence délibérée de grilles réduit le processus à une saisie visuelle immédiate, favorisant des approches expérimentales de la mise en forme sonore.

Architecture & Routage

Pattern Bank
Sélectionnez une cellule vide (A-C) pour initier une séquence.
Ondes
Définit le spectre : ∿ (Sinus), ⊓ (Carrée), ∧ (Scie).
Snap
Force une quantification temporelle absolue sur l'axe X.
FX Matrix
Route les signaux audio vers les modules DSP via les indicateurs de couleur.

Performance & Macro

Poke Me Pad
Interface tactile pour des déclenchements de fréquence immédiats.
X/Y Link
Convertit temporairement le pad en macro-contrôleur d'effets.
Harmonize
Contraint toutes les coordonnées Y à des échelles musicales.
Ext Sync
Synchronise la boucle globale à une DAW externe via MIDI.

Moteurs de Synthèse (Brushes)

Standard
Oscillateur continu à phase (idéal pour le portamento).
Pressure
Épaisseur du trait dépendante de la vélocité.
Particles
Micro-montage stochastique (synthèse granulaire).
Chord
Multiplication harmonique selon des intervalles définis.
Xenakids
Modulation de phase générant des clusters de dissonance.
Rorbach
Contrepoint symétrique (effet miroir sur l'axe Y).
Overtone
Synthèse spectrale additive (ajout de partiels logarithmiques).
DX7 (FM)
Modulation de fréquence à 2 opérateurs (méthode Chowning).
Fractal
Synthétiseur déstabilisé via le rack FRACTAL (Chaos/Wavefolding).
