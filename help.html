Graphic Synthesis Sequencer

1. Initialisierung & Zeichnung

  • Pattern-Zuweisung: Wähle in der Pattern Bank (unten rechts) eine leere Zelle, um eine neue Sequenz zu initiieren.
  • Grafische Eingabe: Die X-Achse repräsentiert die Zeit (t), die Y-Achse die fundamentale Frequenz (f0). Zeichne Vektoren in das Canvas.
  • Werkzeuge: Wähle zwischen Draw (Vektor-Eingabe) und Erase (Vektor-Tilgung).

2. Spur-Architektur (Tracks)

  • Wellenformen: Definiert das harmonische Spektrum. ∿ (Sinus/Grundton), ⊓ (Rechteck/ungerade Obertöne), ∧ (Sägezahn/alle Obertöne).
  • Snap (Quantisierung): Erzwingt eine harte Rasterung der X-Koordinaten zur absoluten Taktsynchronität.
  • Routing: Nutze Solo/Mute für isolierte Audiosignal-Kontrolle und die Slider für die Amplituden-Skalierung.

3. Performative Interaktion

  • Poke Me Pad: Ein taktiles Echtzeit-Interface. Triggert sofortige Frequenz-Zuweisungen auf der selektierten Spur.
  • Effekt-Matrix: Klicke auf die Farbindikatoren, um das Signal in dedizierte DSP-Module zu routen.
  • X/Y Link: Wandelt das Pad in einen Makro-Controller um, der algorithmisch Parameter der Effekt-Engine interpoliert.

4. Taktung & Makro-Struktur

  • Transport: Start/Stopp der globalen Playback-Schleife (Shortcut: Leertaste).
  • Harmonize: Zwingt alle Y-Koordinaten durch mathematische Rundung auf vordefinierte musikalische Skalen.
  • EXT SYNC: Synchronisiert den Sequenzer mit deiner DAW (via virtuellem MIDI-Port).

5. Geometrische Synthese-Werkzeuge (Brushes)

  • Standard: Kontinuierlicher Phasen-Oszillator (perfektes Portamento).
  • Pressure: Geschwindigkeitsabhängige Linienführung. Simuliert haptischen Druck.
  • Calligraphy: Winkel-determinierte Geometrie.
  • Particles: Stochastische Mikromontage (Granularsynthese). Zerlegt die Kurve in asynchrone, granulare Grains.
  • Chord: Harmonische Frequenz-Vervielfachung. Generiert parallele Oszillatoren.
  • Xenakids (Swarm): Unkorrelierte Phasenmodulation. Erzeugt schwebende Dissonanz-Cluster.
  • Rorbach (Rorschach): Achsensymmetrischer Kontrapunkt.
  • Overtone: Additive Spektralsynthese. Erweitert den Grundton um logarithmisch angeordnete Partialtöne.
  • DX7 (FM): Frequenzmodulation nach John Chowning.
  • Fractal: Destabilisierter Synthesizer.
    DSP-Steuerung via "FRACTAL" Rack:
    → CHAOS: Injiziert Rausch-Jitter in die Y-Matrix.
    → MORPH: Aktiviert eine nicht-lineare Transferfunktion (WaveShaper/Wavefolding).

6. Datensicherung & Rendering

  • SAVE: Speichert den Zustand aller Vektor-Matrizen auf ein Pad der Bank.
  • Live Arrangement: Sequenzielles Umschalten zwischen Pattern-Speichern in Echtzeit.
  • Export WAV: Berechnet die Audio-Buffer iterativ (Offline-Rendering).
  • Live REC: Rekordiert den Master-Bus und exportiert improvisierte Performances.
  • Import/Export: Serialisiert die komplette Session (JSON).

Graphic Synthesis Sequencer

1. Initialization & Drawing

  • Assign Pattern: Select an empty cell in the Pattern Bank.
  • Vector Input: The X-axis represents time (t), the Y-axis fundamental frequency (f0).
  • Tools: Toggle between Draw and Erase.

2. Track Architecture

  • Waveforms: Defines harmonic spectrum. ∿ (Sine), ⊓ (Square), ∧ (Sawtooth).
  • Snap (Quantization): Forces absolute temporal synchronization.
  • Routing: Use Solo/Mute and volume sliders.

3. Performative Interaction

  • Poke Me Pad: A tactile real-time interface for instant frequencies.
  • FX Matrix: Route audio signals into dedicated DSP modules.
  • X/Y Link: Hijacks the tactile pad to interpolate FX parameters.

4. Clock & Macro Structure

  • Transport: Start/Stop global playback loop.
  • Harmonize: Constrains all Y-coordinates to predefined musical scales.
  • EXT SYNC: Slaves the sequencer to your DAW.

5. Geometric Synthesis Brushes

  • Standard: Continuous phase oscillator (Portamento).
  • Pressure: Velocity-dependent strokes simulating physical pressure.
  • Calligraphy: Angle-determined geometry.
  • Particles: Stochastic micro-montage (Granular Synthesis).
  • Chord: Harmonic frequency multiplication.
  • Xenakids (Swarm): Uncorrelated phase modulation.
  • Rorbach (Rorschach): Axis-symmetric counterpoint.
  • Overtone: Additive Spectral Synthesis.
  • DX7 (FM): Frequency Modulation (John Chowning method).
  • Fractal: Destabilized synthesizer.
    DSP-Control via "FRACTAL" Rack:
    → CHAOS: Injects noise-jitter into the Y-matrix.
    → MORPH: Activates a non-linear transfer function (Wavefolding).

Séquenceur de Synthèse Graphique

1. Initialisation & Dessin

  • Assignation de Pattern : Sélectionnez une cellule vide dans la Pattern Bank.
  • Entrée Graphique : L'axe X = temps (t), axe Y = fréquence (f0).
  • Outils : Basculez entre Draw et Erase.

2. Architecture des Pistes

  • Formes d'Onde : ∿ (Sinusoïde), ⊓ (Carrée), ∧ (Dents de scie).
  • Snap : Force la synchronisation temporelle absolue sur la grille.
  • Routage : Utilisez Solo/Mute pour isoler le signal.

3. Interaction Performative

  • Poke Me Pad : Interface tactile en temps réel.
  • Matrice d'Effets (FX) : Routez le signal vers des modules DSP.
  • Lien X/Y (Link) : Convertit le pad en macro-contrôleur d'effets.

4. Horloge & Macro-Structure

  • Transport : Démarre ou arrête la boucle.
  • Harmoniser : Contraint la hauteur sur des échelles musicales.
  • EXT SYNC : Synchronise avec votre DAW.

5. Outils de Synthèse (Brushes)

  • Standard : Oscillateur continu (Portamento).
  • Pressure : Tracé dépendant de la vélocité.
  • Calligraphy : Géométrie par angle.
  • Particles : Synthèse Granulaire.
  • Chord : Multiplication harmonique.
  • Xenakids (Swarm) : Modulation de phase (dissonance).
  • Rorbach (Rorschach) : Contrepoint symétrique.
  • Overtone : Synthèse Spectrale Additive.
  • DX7 (FM) : Modulation de Fréquence.
  • Fractal : Synthétiseur déstabilisé.
    Contrôle via "FRACTAL" :
    → CHAOS : Injecte un bruit de phase.
    → MORPH : Wavefolding / distorsion extrême.
